# Snaproof landing page

A single-page brief prepared for Snaproof ahead of eMerge Americas 2026 (booth 951). The page uses Snaproof's own tokens (Inter, indigo #4241ff, soft-gradient hero, rounded cards) so it reads as native to snaproof.it rather than as a vendor handout. It carries a scoped architecture-sprint pitch keyed to Snaproof's "AI venture capital" framing vs. its cryptographic product core, references the ICM paper and the Feeld precedent, and routes to Matt Creamer (CRO) via Calendly. Eduba branding is limited to the footer per sprint rules. Mobile-tested at 375px.
